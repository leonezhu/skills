# 智能任务分配指南

## 核心原则
**复杂任务由 AI 分配，不在 Python 中硬编码**

## 使用方法

### 1. 在周计划中明确描述
```markdown
## 本周任务
-[ ] 读《深度学习》第1-3章并写3篇笔记（分配：每天1章+1笔记）
-[ ] 完成3个设计任务（分配：周一到周三）
-[ ] 写5篇读书笔记（分配：每天1篇）
```

### 2. AI 自动识别并分配
运行 `openskills read daily-plan` 时，AI 会：
- 识别需要分配的任务
- 计算本周日期
- 将任务平均分配到每天
- 生成每日具体任务

### 3. 生成的每日计划
```markdown
## 2025-12-22 (周一)
- [ ] 阅读《深度学习》第1章
- [ ] 写《深度学习》第1篇笔记

## 2025-12-23 (周二)
- [ ] 阅读《深度学习》第2章
- [ ] 写《深度学习》第2篇笔记

## 2025-12-24 (周三)
- [ ] 阅读《深度学习》第3章
- [ ] 写《深度学习》第3篇笔记
```

## AI 识别规则

| 任务描述 | 识别结果 | 分配方式 |
|---------|---------|---------|
| `读《书名》第1-3章` | 阅读任务 | 3天，每天1章 |
| `写3篇笔记` | 写作任务 | 3天，每天1篇 |
| `完成3个任务` | 多个任务 | 3天，每天1个 |
| `读《书名》第1-3章并写3篇笔记` | 阅读+写作 | 3天，每天1章+1篇 |

## 关键词检测
AI 会识别以下关键词：
- `读` + `章` + `篇` → 阅读+写作
- `阅读` + `章` → 阅读
- `写` + `篇` → 写作
- `完成` + `个` → 多个任务
- `分配：` → 明确分配指示

## 最佳实践

### ✅ 推荐
1. **明确描述** - 使用"分配："说明
2. **合理拆分** - 3-5天的任务量
3. **具体化** - 书名、章节、数量

### ❌ 避免
1. **任务过大** - "读完整本书" → 拆分成章节
2. **模糊描述** - "学习Python" → 具体化
3. **跨周任务** - 拆分到多周

## 总结
1. **描述清晰** - 让 AI 理解任务结构
2. **自动拆分** - AI 将任务分配到每天
3. **无需代码** - 通过文档描述实现智能分配
